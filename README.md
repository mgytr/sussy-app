# SUstainability Search and AI Engine - SUSSIE

## Description

SUSIE is a NextJS webapp that for the <b>2nd Global Sustainability Innovation Competition(GSIC)</b> to help consumers idenitify ways to sustainably use the item. This project solves the problem of consumers not knowing how to dispose of recycle named / unnamed items but want to make an impact. This product is ment to reduce waste by properly recycling and encourage reusing products by giving tips and as a way use as guidance for non-specific items that most people are not sure whether to Reuse, Recycle or Dispose of. SUSSIE was made to support sustainability by reducing landfill waste, encouraging reuse over disposal, recovering valuable materials from electronics, lowering carbon emissions, and extending product lifecycles.

## Services and packages used

- NextJS - Backend and frontend
- Gemini - AI that user queries to
- HTTP Webhooks - To collect terlemitty
- Redis - for storing server state

## How to install

Clone this repo

```
git clone
```

CD into SUSSIE-app

```
cd sussie-app
```

To start the app bun

```
npm run dev
```
